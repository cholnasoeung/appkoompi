# Tailwind CSS Journal

List of 20 Tailwind classes I now understand:

1. `bg-blue-600` - Sets background color to blue-600
2. `text-white` - Sets text color to white
3. `p-4` - Adds padding of 1rem on all sides
4. `mx-auto` - Centers the element horizontally with auto margins
5. `flex` - Applies flexbox layout
6. `justify-between` - Distributes space between flex items
7. `items-center` - Centers flex items vertically
8. `space-x-4` - Adds horizontal space between child elements
9. `hover:text-blue-200` - Changes text color on hover
10. `py-20` - Adds vertical padding of 5rem
11. `text-center` - Centers text
12. `text-4xl` - Sets font size to 4xl (3rem)
13. `font-bold` - Makes text bold
14. `mb-4` - Adds bottom margin of 1rem
15. `text-lg` - Sets font size to lg (1.125rem)
16. `mb-8` - Adds bottom margin of 2rem
17. `px-6` - Adds horizontal padding of 1.5rem
18. `py-3` - Adds vertical padding of 0.75rem
19. `rounded` - Applies border radius
20. `transition` - Adds transition effects
